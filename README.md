



## Getting Started

### Clone the Repo

```
git clone git@github.com:Washeem2002/daily-diary.git
npm install
```

### Start the App

Make sure your mongodb is setup.

For preview or production, do:

```
npm start
```

For development, do:

```
npm run dev
```
